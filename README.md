# JobFinder 🔍

A real-time job search web application built using HTML, CSS, JavaScript, and RapidAPI.

Search live jobs across India, filter by job type, and save jobs you like — all in one place.

## 🚀 Features

- Real-time job search
- Job type filtering
- Save jobs using localStorage
- Responsive modern UI
- Fast and simple user experience

## 🛠️ What I Learned

- Working with REST APIs
- Async/Await and error handling
- Using localStorage for data persistence
- Deploying projects using Netlify
- Improving frontend UI/UX

## 💻 Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- RapidAPI
